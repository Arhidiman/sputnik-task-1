Запуск приложения с сервером:
1. Клонируем проекты в отдельные папки с гитхаба (если сервер скопировать в папку с клиентом, то не будет работать)
2. В корне сервера и корне клиента по отдельности делаем npm install
3. Сервер запускается командой node server.js
4. Клиент запускается с помощью npm start
5. Клиент работает на порте 3000, у бэкэнда порт 3011
